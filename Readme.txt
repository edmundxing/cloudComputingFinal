The folders contains the codes and results for midterm at the course of Cloud Computing and Storage.

feature_extraction: the source files for image signature computation.
inverted_index: the source files for inverted indexing.
product_quantization: the source files for product quantization.
results: (1) the randomly-selected retrieval images.
	In each subfolder, the image naming with "0_*.png" represents the query image, those naming
	with "1_*.png", "2_*.png", and "3_*.png" denote top-1, -2, and -3 ranked images, respectively.
	 (2) The three types of image features of randomly-selected images.
	 (3) The statistical analysis of retrieval performance.

screenshots: screen shots of product quantization and distributed inverted indexing.
